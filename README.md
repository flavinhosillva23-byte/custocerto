# Custo Certo 3.0

Inclui página inicial, filtros, pesquisa, página de produto, Nota Custo Certo, assistente de compra e comparação de até quatro TVs.

## Publicar

Substitua os arquivos do repositório pelos arquivos desta pasta e mantenha as pastas `css`, `js` e `sql`.

## Atenção

Os preços e especificações são demonstrativos. Confirme os valores nas lojas. Para atualização real, será necessário integrar APIs, feeds de afiliados ou um painel administrativo conectado ao Supabase.

O arquivo `sql/supabase_schema.sql` contém a nova estrutura inicial do banco.
